Detailed Description

Represents the continent of Antarctica and all Earth-related values.

This class wraps all available low level information (such as BEDMAP, PREM, etc.) and provides uniform methods to access such information. This is the only class that should be used to access Earth-related information.

Unless otherwise stated, distance units are in km, density is and angles are in radians with theta measured from the North Pole, and phi azimuthally from the prime meridian (i.e. phi == lon)
