Detailed Description

Abstract class representing a neutrino.

This abstract class is used to represent a neutrino of any flavor. For functions that can acceps any kind of neutrino, use Neutrino& or Neutrino* as the argument type

Return the primary particle from a neutrino interaction.

For a charged particle interaction, this returns a lepton with an appropriately sampled energy. For a neutral current interaction, this returns a neutrino with the appropriately sampled energy.
